
"# Ejercicio 03: Estructura Backend (Battle Royale)

**Estudiante:** Irma Yaneht Arias Garcia

## Descripción
Diseño de arquitectura backend basada en el patrón MVC (Modelo-Vista-Controlador) para la gestión de un videojuego Battle Royale.

## Estructura de carpetas
- `src/controllers/`: Gestiona las peticiones del usuario.
- `src/services/`: Lógica de negocio pesada (cálculos de partidas).
- `src/models/`: Definición de la estructura de datos.
- `src/routes/`: Definición de los endpoints de la API.

## Razonamiento
Se utilizó una arquitectura modular para separar la lógica de negocio de la gestión de rutas, permitiendo que el proyecto sea escalable y fácil de mantener.
