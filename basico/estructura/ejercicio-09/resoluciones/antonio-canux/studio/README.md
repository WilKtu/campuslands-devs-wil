# Proyecto: Estudio de Tatuajes

Este repositorio contiene la estructura base para la gestión de clientes, diseños y agenda del estudio.

## Política de Privacidad y Manejo de Datos
1. **Confidencialidad:** Toda la información personal de los clientes (nombres, contactos, historial médico) es estrictamente confidencial.
2. **Imágenes Sensibles:** Está prohibido subir fotografías reales de los clientes o sus tatuajes en zonas sensibles sin consentimiento explícito. 
3. **Uso de Placeholders:** Para el entorno de desarrollo y pruebas, se deben utilizar archivos `.txt` o imágenes genéricas de *placeholder* para representar las fotos de referencia corporal de los clientes.
