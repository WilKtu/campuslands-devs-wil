# Documentación: Nomenclatura de Portadas (books/covers/)

Para estandarizar y facilitar la vinculación entre los datos y las imágenes de manera programática, los archivos de las portadas deben seguir esta regla de nomenclatura:

**Formato:** `[book_id]_[titulo_simplificado].[extension]`

**book_id:** El identificador único del libro asignado en su archivo JSON (ej. bk_001).

**titulo_simplificado:** El título del libro en minúsculas, sin espacios (usar guiones bajos _) y sin caracteres especiales o tildes.

**extension:** Formatos permitidos .jpg, .png o .webp.

### Ejemplos válidos:

- bk_001_dune.jpg

- bk_002_fundacion.png

- bk_003_el_problema_de_los_tres_cuerpos.webp
