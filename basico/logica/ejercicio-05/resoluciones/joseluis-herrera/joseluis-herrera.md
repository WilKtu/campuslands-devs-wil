# Futbol Sala 

## Explicacion 
Se realizo una solucion para un equipo de futbol sala en el cual se realizaron las sigueintes funciones calcular los puntos por equipo, la diferencia de goles entre equipos y orden entre los equipos. 

## Como pense en la solucion 
Primero lei el documento, luego empece a hacer el codigo paso a paso ver que es lo que me pedian como por ejemplo un par de grupos entonces decidi hacer un array de equipo, luego decidi recorrer ese array de grupo para calcular los puntos con un foreach, ademas de eso para ver la diferencia entre los equipos lo hice con sort, para al final mostrarse de mayor a menor con un foreach igual.
